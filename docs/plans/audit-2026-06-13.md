## BACKEND FINDINGS

### Error Handling / Job Failure Paths

| Severity | File | Issue |

|---|---|---|

| LOW | `api/chat.py:109-135` | `send_message` returns `StreamingResponse` — no Pydantic model, not in OpenAPI schema |
